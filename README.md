# Limiter-la-charge-80-d-un-v-hicule-electrique-Stellantis
Si votre intégration véhicule (Stellantis, ou autre marque) ne remonte le niveau de batterie que toutes les 15-30 minutes, il devient impossible de calculer une heure de fin de charge précise ou de couper au bon moment. Ce projet règle ce problème sans dépendre du cloud.
